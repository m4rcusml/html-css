Pseudo Classe normalmente diz respeito a estados de elementos, no caso de links, por exemplo:

  • a:hover = diz o que acontece quando passamos o mouse em cima do link (ou qualquer outro elemento).
  • a:visited = diz o que acontece quando um link já foi visitado.
  • a:active = diz o que acontece quando um link (e provavelmente qualquer outro elemento) é clicado.
