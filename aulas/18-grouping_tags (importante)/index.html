O que são?

Grouping Tags são tags especiais do HTML que separam o conteúdo do site de forma semântica. Ao invés de criar várias div's e identificar elas com id ou class usamos essas div's ou boxes especiais do HTML e elas são:

Header:
Box que indica o cabeçalho do site.
Nav:
Box que indica o menu do site.
Main:
Box que indica o conteúdo principal do site.
Section:
Box que indica uma seção qualquer, seja de conteúdo ou qualquer outra coisa que precise estar no site, normalmente é identificada dependendo do propósito (olhar o código-fonte).
Article:
Box que indica um artigo, sei lá.
Aside:
Box que indica sei lá cara é um aside, nesse site eu usei o aside pra escrever esse "Aprendi com: Gustavo Guanabara" kk não sei explicar.
